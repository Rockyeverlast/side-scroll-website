This is a Demo for a side scrolling website

I built this site with Gsap side scrolling tween from their codepen demo and used some customization to add navigation. I wanted to break from the traditional vertical scrolling websites to a side scroll so that the web page can use the full screen for the user's viewing pleasure.
I also used the page color transition as a way to direct the user viewing the page to the necessary content. This website is still a work in progress as it is not mobile responsive yet as the target is a desktop or a laptop screen size.
